Source

Assessory is open source, under the MIT Licence. The source-code is available on GitHub

Tech stack

Assessory is built using

  • Scala and Scala.js as the programming language
  • React.js (via scalajs-react) for the UI
  • Play Framework for the server
  • MongoDB for the database

This keeps it all fairly compact, performant, and makes it less likely to break even though the coding is done sporadically.

History

The very first version was built at NICTA for use in a 200-student software studio class at the University of Queensland. In 2015, Will moved to the University of New England and found he needed it even more for online classes.
